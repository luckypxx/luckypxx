#正文

### 抽象

抽象是指对现实世界问题和实体的本质表现，行为，特征进行建模

### 对象的三要素

id、type、value

### 对象的三大特性

封装、继承、多态

### 封装和接口

**封装**

封装是对数据和信息进行隐藏，对象的使用者只能通过接口来访问或修改对象的数据

### 继承

所有的类都继承于根类(object)

**根类**

	class Animal(object):

		#静态变量

	    x=0 

		#非静态成员方法

		def Print(self):

			print "print" 

		#非静态变量

		def __init__(self):

			self.Name="name"

			self.Age=10
		
		#静态成员方法

		@staticmethod

		def Print1()

			print "print1"

	print Animal.x

	#类的实例化

	A=Animal()

	print A.Name,A.Age

	

根类中有静态变量、非静态成员方法，非静态变量、静态成员方法

非静态变量和方法要有self关键字

### 继承

	class Dog(Animal)

Dog继承自Animal

### 多态

子类可以进行自由扩展，对调用封闭
